# Create dataframes for all data we are reviewing

# Create a dataframe of developers and dropping any rows where CompanyType is NaN
developer_df = developer_df.dropna(subset = ['CompanyType'])

# Fill NaNs in the JobSatisfaction and CareerSatisfaction columns with the mean
developer_df['JobSatisfaction'].fillna(developer_df['JobSatisfaction'].mean(), inplace = True)
developer_df['CareerSatisfaction'].fillna(developer_df['CareerSatisfaction'].mean(), inplace = True)
# Convert columns to integers
developer_df['JobSatisfaction'] = developer_df['JobSatisfaction'].astype(int)
developer_df['CareerSatisfaction'] = developer_df['CareerSatisfaction'].astype(int)

# Create a dataframe for the auditory environment and removing NaNs
auditory_df = developer_df.dropna(subset = ['AuditoryEnvironment'])

# Create a datafram for how often a developer works from home
remote_work_df = developer_df.dropna(subset = ['HomeRemote'])

I created 3 dataframes for the different groups to answer the different questions. For my developer dataframe, I dropped all rows where the respondent did not select a company type. I decided to leave the answers, "I don't know", "I prefer not to say", and "Something else" because these answers may be valuable in the analysis. I did not want to remove too many rows, I decided to replace any NaNs with the mean values for both Job and Career Satisfaction. I then converted those responses to integers to assist with analysis.¶

For the auditory dataframe and remote work dataframe, I decided that I would drop any row where the respondent did not identify those variables.¶

# Sort the average values highest to lowest
developer_group_new = developer_group_new.sort_values('average', ascending = False)
